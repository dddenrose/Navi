# 風險管理基礎 — 停損、部位控制、分散投資

## 概述

風險管理是投資中最重要但最常被忽略的環節。好的風險管理不是為了避免虧損，而是為了確保在虧損發生時，損失在可承受範圍內，讓你有資本繼續參與市場。

> 📌 **核心原則**：先想可能虧多少，再想可能賺多少。

---

## 1. 停損 (Stop Loss)

### 定義

預先設定一個價格水準，當股價跌到該水準時執行賣出，限制單筆交易的最大虧損。

### 停損方法

#### 固定百分比停損

設定固定的虧損比例作為停損點，例如：-5%、-8%、-10%。

- **優點**：簡單明確
- **缺點**：不考慮個股波動特性

#### 技術面停損

根據技術分析的關鍵價位設定停損：

- 跌破前波低點
- 跌破重要均線（如 20MA、60MA）
- 跌破支撐線或趨勢線
- 跌破布林通道下軌

#### ATR 停損（波動率停損）

使用 ATR（Average True Range，平均真實波幅）設定停損：

$$
停損價 = 進場價 - N \times ATR
$$

- 常用 $N = 1.5 \sim 3$
- 優點：自動適應不同波動性的股票

### 停損紀律

- **進場前就決定停損價**，不要買了之後再想
- **嚴格執行**，不要因為「再等等看」而移動停損
- **停損不代表判斷錯誤**，停損是風險管理的一部分

## 2. 部位控制 (Position Sizing)

### 核心概念

決定每筆交易投入多少資金，這比選股和擇時更重要。

### 固定金額法

每筆交易投入固定金額。例如：總資金 100 萬，每筆投入 10 萬。

- 優點：簡單
- 缺點：不考慮風險大小

### 固定比例風險法（推薦）

每筆交易承受的最大虧損不超過總資金的固定比例。

$$
部位大小 = \frac{\text{總資金} \times \text{風險比例}}{\text{每股風險（進場價 - 停損價）}}
$$

**範例**：

- 總資金：100 萬
- 風險比例：2%（每筆最多虧 2 萬）
- 買入價：50 元，停損價：45 元（每股風險 5 元）
- 部位大小 = 20,000 ÷ 5 = 4,000 股

### Kelly Criterion（凱利公式）

$$
f^* = \frac{bp - q}{b}
$$

其中：

- $f^*$：最佳投注比例
- $b$：盈虧比（平均獲利 ÷ 平均虧損）
- $p$：勝率
- $q$：敗率（$1 - p$）

> ⚠️ 凱利公式的完整比例過於激進，實務上建議使用 **半凱利**（Half Kelly）或更保守的比例。

### 常見風險參數

| 類型   | 單筆風險比例 | 適用               |
| ------ | ------------ | ------------------ |
| 保守型 | 0.5-1%       | 長線投資人         |
| 穩健型 | 1-2%         | 一般投資人（推薦） |
| 積極型 | 2-3%         | 有經驗的交易者     |
| 激進型 | > 5%         | 不建議             |

## 3. 分散投資 (Diversification)

### 核心理念

「不要把所有雞蛋放在同一個籃子裡。」

透過分散投資來降低非系統性風險（個股風險），只承擔系統性風險（市場風險）。

### 分散層次

#### 個股分散

- 建議持有 **10-20 檔**股票
- 單一個股佔比不超過 **10-15%**
- 過度分散（> 30 檔）反而稀釋報酬，且難以追蹤

#### 產業分散

- 不要重壓單一產業，即使再看好
- 同一產業的股票常有高度相關性
- 例：同時持有台積電和聯發科，分散效果有限

#### 資產類別分散

| 資產   | 特性     | 與股票的相關性 |
| ------ | -------- | -------------- |
| 債券   | 穩定收益 | 低或負相關     |
| 黃金   | 避險資產 | 通常負相關     |
| 現金   | 無風險   | 零相關         |
| 不動產 | 通膨保護 | 中等相關       |

#### 時間分散（分批進場）

- **定期定額**：每月固定投入，降低擇時風險
- **分批佈局**：將預計投入的資金分 3-5 批，在不同時間點進場
- 避免一次All-in 在最高點

## 4. 風險報酬比 (Risk-Reward Ratio)

### 定義

$$
風險報酬比 = \frac{\text{預期獲利}}{\text{預期虧損}} = \frac{\text{目標價} - \text{進場價}}{\text{進場價} - \text{停損價}}
$$

### 判斷標準

| 風險報酬比 | 評估                           |
| ---------- | ------------------------------ |
| < 1:1      | 不值得交易                     |
| 1:1        | 損益平衡，需高勝率才有正期望值 |
| 2:1        | 良好，勝率 40% 就能獲利        |
| 3:1 以上   | 優秀，可容忍較低勝率           |

### 期望值公式

$$
期望值 = (勝率 \times 平均獲利) - (敗率 \times 平均虧損)
$$

只有期望值 > 0 的策略才值得長期執行。

---

## 5. 最大回撤 (Maximum Drawdown)

### 定義

從資產淨值的歷史最高點到最低點的最大跌幅。

$$
MDD = \frac{\text{谷底值} - \text{峰頂值}}{\text{峰頂值}} \times 100\%
$$

### 回撤與回本

| 跌幅 | 回本所需漲幅 |
| ---- | ------------ |
| -10% | +11.1%       |
| -20% | +25.0%       |
| -30% | +42.9%       |
| -50% | +100.0%      |
| -70% | +233.3%      |

> 這就是為什麼**控制虧損**比追求獲利更重要：虧得越多，回本越難。

---

## 綜合建議

1. **永遠設停損**：進場前就決定停損位，絕不心存僥倖
2. **控制部位**：單筆風險不超過總資金 2%
3. **分散持股**：10-20 檔，跨產業配置
4. **風險報酬 > 2:1**：只做值得的交易
5. **保留現金**：不要滿倉，保留 10-30% 現金作為彈藥
6. **記錄與檢討**：建立交易日誌，定期檢視風險管理是否落實
