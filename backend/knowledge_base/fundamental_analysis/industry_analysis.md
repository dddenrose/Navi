# 產業分析框架 (Industry Analysis Framework)

## 定義

產業分析是基本面分析的重要環節，透過了解公司所處的產業環境、競爭格局與發展趨勢，評估公司的長期成長潛力與競爭優勢。在選股時，產業分析往往比個股分析更為重要——「選對產業比選對個股更重要」。

## 分析層次

產業分析採用由上而下（Top-Down）的方法：

$$
\text{總體經濟} \rightarrow \text{產業趨勢} \rightarrow \text{個股分析}
$$

---

## 一、產業生命週期

每個產業都會經歷不同的發展階段，不同階段的投資策略截然不同。

| 階段   | 特徵                                       | 投資策略                           |
| ------ | ------------------------------------------ | ---------------------------------- |
| 萌芽期 | 技術剛起步，收入低，虧損多，前景不確定     | 高風險高報酬，適合風險承受力高的人 |
| 成長期 | 營收快速成長，市場滲透率提升，競爭者增加   | 最佳投資階段，成長性佳             |
| 成熟期 | 成長趨緩，市佔率穩定，利潤率高，現金流穩定 | 適合價值投資，關注股息             |
| 衰退期 | 需求下降，競爭者退出，營收與利潤下滑       | 避開或等待產業轉型機會             |

> 📌 **關鍵**：辨別產業處於哪個階段，是做出正確投資決策的第一步。

## 二、波特五力分析 (Porter's Five Forces)

麥可・波特提出的產業競爭分析框架，用於評估產業的競爭強度和獲利潛力。

### 五種競爭力

| 力量             | 分析重點                               | 強度高 → 產業獲利 |
| ---------------- | -------------------------------------- | ----------------- |
| 現有競爭者的對抗 | 同業家數、成長率、差異化程度、退出障礙 | 獲利壓縮          |
| 潛在進入者的威脅 | 進入門檻（資本、技術、規模、法規）     | 利潤可能被瓜分    |
| 替代品的威脅     | 替代品的性能/價格比、轉換成本          | 定價能力受限      |
| 供應商的議價力   | 供應商集中度、轉換成本、原料獨特性     | 成本上升          |
| 買方的議價力     | 客戶集中度、轉換成本、產品差異化程度   | 售價受壓          |

### 應用範例

以台灣半導體業為例：

- **現有競爭者**：全球僅少數先進製程廠商 → 競爭強度中
- **進入門檻**：極高（千億資本、技術壁壘）→ 新進者威脅低
- **替代品**：短期內無替代 → 威脅低
- **供應商議價力**：設備廠有一定議價力（如 ASML 壟斷 EUV）→ 中高
- **買方議價力**：大客戶議價力高（如 Apple）→ 中高

→ **結論**：進入門檻極高，但大客戶有議價力，整體獲利能力中上

## 三、產業上下游分析

### 供應鏈結構

$$
\text{上游（原料/零件）} \rightarrow \text{中游（製造/組裝）} \rightarrow \text{下游（品牌/通路/終端）}
$$

### 分析要點

| 面向     | 分析問題                                       |
| -------- | ---------------------------------------------- |
| 上游     | 原料是否供給充足？價格趨勢如何？是否有壟斷？   |
| 中游     | 製造技術門檻高低？產能利用率？毛利率趨勢？     |
| 下游     | 終端需求是否成長？品牌忠誠度？通路掌控力？     |
| 垂直整合 | 公司是否跨足上下游？整合帶來成本優勢還是風險？ |

### 觀察指標

- **上游漲價**：中下游成本提升 → 注意毛利率壓縮
- **下游需求轉弱**：拉貨動能下降 → 注意營收衰退
- **庫存水位**：過高代表需求不振，過低可能帶來急單效應

## 四、競爭優勢（護城河）分析

巴菲特強調「投資有護城河的公司」。護城河是指公司能夠長期維持競爭優勢的結構性因素。

### 五種護城河

| 護城河類型  | 說明                         | 台股案例               |
| ----------- | ---------------------------- | ---------------------- |
| 成本優勢    | 規模經濟、學習曲線帶來低成本 | 台積電（先進製程規模） |
| 網路效應    | 使用者越多，產品價值越高     | LINE（通訊軟體）       |
| 品牌價值    | 品牌帶來定價能力與客戶忠誠   | 統一超（7-ELEVEN）     |
| 轉換成本    | 客戶切換到競爭對手的代價高   | 企業 ERP 系統          |
| 法規/特許權 | 法律保護、執照壁壘           | 中華電信（電信執照）   |

### 護城河強度判斷

- **寬護城河**：多種護城河同時存在，競爭優勢能持續 10 年以上
- **窄護城河**：有一定優勢但可能在 5 年內被侵蝕
- **無護城河**：產品同質性高，價格競爭激烈

## 五、產業關鍵指標

不同產業需要關注的核心營運指標不同：

| 產業     | 關鍵指標                                                      |
| -------- | ------------------------------------------------------------- |
| 半導體   | 產能利用率、ASP（平均售價）、製程節點、設計接單（Design Win） |
| 電子代工 | 毛利率、產能利用率、客戶集中度、自動化率                      |
| 金融保險 | ROE、淨利差（NIM）、逾放比、資本適足率                        |
| 零售通路 | 同店營收成長、坪效、展店速度、庫存週轉天數                    |
| 電信     | ARPU（用戶平均營收）、用戶流失率、5G 滲透率                   |
| 營建     | 預售金額、存貨（土地庫存）、推案量、利率敏感度                |
| 生技醫藥 | 研發管線、臨床試驗進度、藥證取得、授權金                      |
| 觀光餐飲 | RevPAR（每房營收）、翻桌率、來客數、客單價                    |

## 六、產業景氣循環

### 景氣循環位置判斷

| 指標                 | 上行期       | 下行期           |
| -------------------- | ------------ | ---------------- |
| 營收年增率           | 正成長且加速 | 成長趨緩或負成長 |
| 毛利率               | 上升         | 下降             |
| 存貨水位             | 合理或偏低   | 過高             |
| 產能利用率           | > 80%        | < 70%            |
| 產業投資（資本支出） | 增加         | 減少             |

### 循環型 vs 非循環型產業

| 類型     | 特徵                     | 代表產業                   |
| -------- | ------------------------ | -------------------------- |
| 循環型   | 與經濟景氣高度相關       | 鋼鐵、塑化、航運、面板     |
| 非循環型 | 需求穩定不受景氣大幅影響 | 食品、電信、公用事業、醫療 |

> 📌 **投資建議**：循環股適合在景氣谷底佈局，PE 最高時反而是買點（盈利谷底）；PE 最低時可能是賣點（盈利高峰）——與一般直覺相反。

## 實戰分析步驟

1. **定位產業生命週期**：判斷是成長期還是成熟期
2. **評估競爭格局**：五力分析，找出產業獲利的主要驅動因素
3. **追蹤上下游**：了解供應鏈動態，找出領先指標
4. **辨識護城河**：目標公司有什麼持久的競爭優勢
5. **監測景氣循環**：目前處於循環的哪個位置
6. **比較產業內個股**：在同一產業中選擇最有優勢的標的

## 限制

- **產業分類模糊**：許多公司跨足多個產業，難以單一分類
- **變化速度快**：科技業的產業生命週期越來越短
- **資訊不對稱**：產業深度資訊通常需要專業報告才能取得
- **黑天鵝事件**：地緣政治、法規變動等無法用模型預測
