# MACD — 指數平滑異同移動平均線

## 定義

MACD（Moving Average Convergence Divergence）由 Gerald Appel 於 1979 年發明，是一種趨勢跟蹤動量指標，透過兩條指數移動平均線（EMA）之間的關係來判斷趨勢方向與動能。

## 組成元素

MACD 由三個部分組成：

| 元素                 | 全名           | 計算方式          |
| -------------------- | -------------- | ----------------- |
| DIF（快線）          | Difference     | EMA(12) − EMA(26) |
| DEA / Signal（慢線） | Signal Line    | DIF 的 EMA(9)     |
| 柱狀圖（Histogram）  | MACD Histogram | DIF − DEA         |

## 計算公式

### 步驟一：計算 EMA

$$
EMA_t = P_t \times k + EMA_{t-1} \times (1 - k)
$$

其中 $k = \frac{2}{N+1}$，$N$ 為週期。

### 步驟二：計算 DIF

$$
DIF = EMA_{12} - EMA_{26}
$$

### 步驟三：計算 DEA

$$
DEA = DIF \text{ 的 } EMA(9)
$$

### 步驟四：計算柱狀圖

$$
Histogram = DIF - DEA
$$

> 有些平台將柱狀圖乘以 2 以放大顯示，使用時請注意。

## 判讀方法

### 金叉與死叉

- **金叉（Golden Cross）**：DIF 由下往上穿越 DEA → 看多信號
  - 零軸之上的金叉：強多信號
  - 零軸之下的金叉：弱多信號（可能只是反彈）
- **死叉（Death Cross）**：DIF 由上往下穿越 DEA → 看空信號
  - 零軸之下的死叉：強空信號
  - 零軸之上的死叉：弱空信號（可能只是回檔）

### 零軸分析

- **DIF 與 DEA 在零軸之上**：多頭市場
- **DIF 與 DEA 在零軸之下**：空頭市場
- **DIF 上穿零軸**：趨勢由空轉多
- **DIF 下穿零軸**：趨勢由多轉空

### 柱狀圖判讀

- **紅柱增長**：多頭動能增強
- **紅柱縮短**：多頭動能減弱（可能即將轉折）
- **綠柱增長**：空頭動能增強
- **綠柱縮短**：空頭動能減弱（可能即將反彈）

### 背離

MACD 背離是重要的反轉信號：

- **頂背離**：股價創新高，但 MACD 的 DIF 或柱狀圖未同步創新高 → 上漲動能衰退，可能反轉向下
- **底背離**：股價創新低，但 MACD 的 DIF 或柱狀圖未同步創新低 → 下跌動能衰退，可能反轉向上

> 📌 **實戰注意**：一次背離不可靠，連續兩次以上的背離信號較為可信。

## 參數設定

| 參數       | 標準值 | 變化        |
| ---------- | ------ | ----------- |
| 快線 EMA   | 12     | 短線可用 8  |
| 慢線 EMA   | 26     | 短線可用 17 |
| Signal EMA | 9      | 較少變動    |

- **縮短參數**（如 8, 17, 9）：更敏感、信號更多，但假信號也更多
- **延長參數**（如 19, 39, 9）：更平滑、信號更少，適合中長線

## 使用場景與限制

### 適用場景

- **趨勢確認**：判斷目前是多頭還是空頭
- **進出場時機**：利用金叉、死叉做買賣信號
- **動量評估**：觀察柱狀圖變化評估趨勢強度
- **背離警示**：在趨勢末端提前預警

### 限制

- **滯後性**：MACD 以均線為基礎，天然具有滯後性
- **盤整行情失效**：在窄幅盤整時，金叉死叉頻繁出現，容易被來回洗刷
- **無法判斷超買超賣**：MACD 沒有固定上下界，需搭配 RSI 等振盪指標

### 搭配使用

- **RSI**：搭配判斷超買超賣，過濾假信號
- **KD 隨機指標**：短線交叉確認
- **成交量**：金叉/死叉是否伴隨放量，驗證信號有效性
- **均線系統**：MACD 金叉 + 均線多頭排列 = 強勢做多信號
