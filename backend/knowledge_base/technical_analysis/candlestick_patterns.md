# K 線型態學 (Candlestick Patterns)

## 定義

K 線（又稱蠟燭線、陰陽線）由日本米商本間宗久於 18 世紀發明，是目前最普遍使用的價格圖表。每根 K 線記錄了特定時間內的開盤價、最高價、最低價與收盤價（OHLC），透過 K 線的組合型態可判斷市場多空力道的消長。

## K 線基本結構

| 組成         | 說明                       |
| ------------ | -------------------------- |
| 實體         | 開盤價與收盤價之間的區域   |
| 上影線       | 最高價與實體上緣之間的線段 |
| 下影線       | 最低價與實體下緣之間的線段 |
| 紅 K（陽線） | 收盤價 > 開盤價，代表上漲  |
| 黑 K（陰線） | 收盤價 < 開盤價，代表下跌  |

> 📌 **實體越長**，代表多空某方力道越強；**影線越長**，代表曾經有力道但被收回。

## 單根 K 線型態

### 反轉型態

#### 鍾子線（Hammer）

- **外觀**：小實體 + 長下影線（下影線 ≥ 實體 2 倍），無上影線或極短
- **出現位置**：下跌趨勢末端
- **意義**：空方一度壓低價格，但多方強力收回 → 底部反轉信號
- **確認**：隔日收紅 K 確認

#### 吊人線（Hanging Man）

- **外觀**：與錘子線相同
- **出現位置**：上漲趨勢末端
- **意義**：盤中出現大量賣壓雖被收回，但顯示空方力量增強 → 頂部警訊
- **確認**：隔日收黑 K 確認

#### 倒鍾子線（Inverted Hammer）

- **外觀**：小實體 + 長上影線，無下影線或極短
- **出現位置**：下跌趨勢末端
- **意義**：多方嘗試拉高雖未果，但空方力量減弱 → 潛在底部信號
- **確認**：隔日開高確認

#### 射擊之星（Shooting Star）

- **外觀**：與倒錘子線相同
- **出現位置**：上漲趨勢末端
- **意義**：多方拉高後被空方壓回 → 頂部反轉信號
- **確認**：隔日開低確認

#### 十字線（Doji）

- **外觀**：開盤價 ≈ 收盤價，實體極小或無
- **意義**：多空力道均衡，猶豫不決
- **變體**：
  - **長腿十字**：上下影線都長 → 多空激烈拉鋸
  - **蜻蜓十字**：僅有下影線 → 在低部出現為反轉信號
  - **墓碑十字**：僅有上影線 → 在頭部出現為反轉信號
- **關鍵**：十字線出現在趨勢末端才有反轉意義，盤整中的十字線意義不大

### 持續型態

#### 大陽線（Marubozu 紅）

- **外觀**：長紅實體，無影線或影線極短
- **意義**：多方完全主導，強烈看多

#### 大陰線（Marubozu 黑）

- **外觀**：長黑實體，無影線或影線極短
- **意義**：空方完全主導，強烈看空

## 雙根 K 線組合

### 吞噬型態（Engulfing）

#### 多頭吞噬（Bullish Engulfing）

- **外觀**：第一根為黑 K，第二根為紅 K 且實體完全覆蓋第一根
- **出現位置**：下跌趨勢末端
- **意義**：多方力道強勁逆轉 → 看漲反轉信號
- **可靠度**：高（尤其第二根帶量時）

#### 空頭吞噬（Bearish Engulfing）

- **外觀**：第一根為紅 K，第二根為黑 K 且實體完全覆蓋第一根
- **出現位置**：上漲趨勢末端
- **意義**：空方力道強勁逆轉 → 看跌反轉信號

### 貫穿線與烏雲蓋頂

#### 貫穿線（Piercing Line）

- **外觀**：第一根大黑 K，第二根開低但收盤價超過第一根實體的中點以上
- **意義**：買方在更低位進場拉回 → 底部反轉信號

#### 烏雲蓋頂（Dark Cloud Cover）

- **外觀**：第一根大紅 K，第二根開高但收盤價跌破第一根實體的中點以下
- **意義**：賣方在更高位出場壓低 → 頂部反轉信號

### 母子型態（Harami）

#### 多頭母子（Bullish Harami）

- **外觀**：第一根大黑 K，第二根小紅 K 且完全被第一根實體包含
- **意義**：下跌動能減弱 → 潛在底部信號（需確認）

#### 空頭母子（Bearish Harami）

- **外觀**：第一根大紅 K，第二根小黑 K 且完全被第一根實體包含
- **意義**：上漲動能減弱 → 潛在頭部信號（需確認）

## 三根 K 線組合

### 晨星（Morning Star）

- **組成**：大黑 K → 小實體（跳空向下）→ 大紅 K（收回超過第一根一半）
- **意義**：經典底部反轉型態，可靠度高
- **確認**：第三根帶量更佳

### 夜星（Evening Star）

- **組成**：大紅 K → 小實體（跳空向上）→ 大黑 K（跌回超過第一根一半）
- **意義**：經典頂部反轉型態，可靠度高

### 三白兵（Three White Soldiers）

- **組成**：連續三根紅 K，每根開盤在前一根實體內，收盤創新高
- **意義**：多方持續進攻，強烈看多信號
- **注意**：若第三根出現長上影線，可能為力竭信號

### 三黑鴉（Three Black Crows）

- **組成**：連續三根黑 K，每根開盤在前一根實體內，收盤創新低
- **意義**：空方持續壓制，強烈看空信號

## 型態可靠度排行

| 排名 | 反轉型態        | 可靠度 |
| ---- | --------------- | ------ |
| 1    | 晨星 / 夜星     | ★★★★★  |
| 2    | 吞噬型態        | ★★★★☆  |
| 3    | 三白兵 / 三黑鴉 | ★★★★☆  |
| 4    | 錘子 / 射擊之星 | ★★★☆☆  |
| 5    | 十字線          | ★★★☆☆  |
| 6    | 母子型態        | ★★☆☆☆  |

> 📌 **所有 K 線型態都需要後續確認**，單一型態不構成完整的交易依據。

## 實戰應用原則

### 提高 K 線判讀準確度的方法

1. **結合趨勢方向**：反轉型態必須出現在明確的上升或下降趨勢之後才有意義
2. **觀察出現位置**：在重要支撐/壓力位、均線附近出現的型態更有效
3. **配合成交量**：放量的型態比縮量的更可靠
4. **等待確認**：型態出現後至少等一根 K 線確認再行動
5. **時間框架**：週線圖的型態比日線圖更可靠，日線圖比分鐘圖更可靠

### 限制

- **主觀性高**：同一根 K 線，不同人可能有不同解讀
- **需搭配上下文**：孤立的 K 線型態沒有意義，必須放在趨勢中判讀
- **回測勝率有限**：單一 K 線型態的統計勝率通常只有 50-60%
- **不適用於跳空受限的市場**：台股有漲跌停限制，某些跳空型態較難出現
