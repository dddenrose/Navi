# RSI — 相對強弱指標 (Relative Strength Index)

## 定義

RSI 是由 J. Welles Wilder Jr. 於 1978 年提出的動量振盪指標，用於衡量價格變動的速度與幅度。RSI 的數值介於 0 到 100 之間，是技術分析中最常用的指標之一。

## 計算公式

### 基本公式

$$
RSI = 100 - \frac{100}{1 + RS}
$$

其中：

$$
RS = \frac{\text{平均漲幅（N 日）}}{\text{平均跌幅（N 日）}}
$$

### 計算步驟

1. 選擇時間週期（通常為 14 日）
2. 計算每日收盤價變動（今日收盤 − 昨日收盤）
3. 將正數變動加總除以 N，得平均漲幅
4. 將負數變動取絕對值加總除以 N，得平均跌幅
5. RS = 平均漲幅 ÷ 平均跌幅
6. 代入公式得 RSI

### 平滑 RSI（Wilder's Smoothing）

初次計算後，後續採用指數平滑法：

$$
\text{平均漲幅}_t = \frac{(\text{前一日平均漲幅} \times 13) + \text{今日漲幅}}{14}
$$

## 判讀方法

### 超買與超賣

| RSI 區間      | 意義   | 交易信號                     |
| ------------- | ------ | ---------------------------- |
| RSI > 70      | 超買區 | 可能即將回檔，考慮減碼或觀望 |
| 30 < RSI < 70 | 中性區 | 趨勢持續中，無極端信號       |
| RSI < 30      | 超賣區 | 可能即將反彈，考慮進場或加碼 |

> ⚠️ **注意**：在強趨勢中，RSI 可能長期維持在超買或超賣區。單純以 70/30 做買賣判斷容易過早出場或進場。

### 中心線交叉

- RSI 由下往上穿越 50：偏多信號，上漲動能增強
- RSI 由上往下穿越 50：偏空信號，下跌動能增強

### 背離（Divergence）

背離是 RSI 最有價值的信號之一：

- **看多背離（Bullish Divergence）**：股價創新低，但 RSI 未創新低 → 下跌動能減弱，可能反轉向上
- **看空背離（Bearish Divergence）**：股價創新高，但 RSI 未創新高 → 上漲動能減弱，可能反轉向下

### 失敗擺盪（Failure Swing）

- **看多失敗擺盪**：RSI 跌破 30 → 反彈 → 再次回落但未跌破前低 → 突破反彈高點 → 買入信號
- **看空失敗擺盪**：RSI 突破 70 → 回落 → 再次上升但未超過前高 → 跌破回落低點 → 賣出信號

## 參數設定

| 參數   | 常用值 | 說明                                |
| ------ | ------ | ----------------------------------- |
| 週期   | 14     | Wilder 建議的標準值，適合大多數情況 |
| 短週期 | 7-9    | 更敏感，適合短線交易                |
| 長週期 | 21-25  | 更平滑，適合中長線投資              |

## 使用場景與限制

### 適用場景

- **盤整市場**：RSI 在超買超賣之間擺盪時效果最佳
- **確認趨勢強度**：配合均線使用，確認趨勢是否過熱
- **找尋背離信號**：在趨勢末端識別可能的反轉點

### 限制

- **強趨勢中失效**：在單邊上漲或下跌行情中，RSI 可能長期處於超買/超賣區
- **假背離**：背離信號不代表立即反轉，可能需要多次確認
- **滯後性**：RSI 是基於歷史價格計算的，屬於落後指標

### 搭配使用

建議搭配以下指標使用，互相驗證：

- **MACD**：確認動量方向
- **布林通道**：確認價格相對位置
- **成交量**：驗證價格變動的有效性
- **均線**：判斷大趨勢方向
