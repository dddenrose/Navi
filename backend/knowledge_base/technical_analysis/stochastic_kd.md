# KD — 隨機指標 (Stochastic Oscillator)

## 定義

KD 指標由 George Lane 於 1950 年代提出，是一種動量振盪指標，透過比較當前收盤價在一段期間內最高價與最低價之間的相對位置，來衡量價格動能。KD 值介於 0 ~ 100 之間。

## 組成元素

| 元素               | 全名                 | 說明                            |
| ------------------ | -------------------- | ------------------------------- |
| RSV (未成熟隨機值) | Raw Stochastic Value | 當前價格在 N 日區間中的相對位置 |
| K 線（快線）       | %K                   | RSV 的平滑移動平均              |
| D 線（慢線）       | %D                   | K 值的平滑移動平均              |

## 計算公式

### 步驟一：計算 RSV

$$
RSV = \frac{C - L_n}{H_n - L_n} \times 100
$$

其中：

- $C$ = 當日收盤價
- $L_n$ = 最近 N 日的最低價
- $H_n$ = 最近 N 日的最高價

### 步驟二：計算 K 值

$$
K_t = K_{t-1} \times \frac{2}{3} + RSV_t \times \frac{1}{3}
$$

> 初始 K 值通常設為 50。

### 步驟三：計算 D 值

$$
D_t = D_{t-1} \times \frac{2}{3} + K_t \times \frac{1}{3}
$$

> 初始 D 值通常設為 50。

### 常用參數

| 參數         | 常用值 | 說明                          |
| ------------ | ------ | ----------------------------- |
| RSV 週期 (N) | 9      | 原始設定，適合短線交易        |
| K 平滑係數   | 1/3    | 台股常見設定，又稱「慢速 KD」 |
| D 平滑係數   | 1/3    | 台股常見設定                  |

> 📌 **快速 vs 慢速 KD**：直接以 RSV 當作 K 線稱為「快速 KD」，經過平滑處理的稱為「慢速 KD」。台股慣用慢速 KD（平滑係數 1/3），較能過濾雜訊。

## 判讀方法

### 超買與超賣

| KD 區間     | 意義   | 交易信號                     |
| ----------- | ------ | ---------------------------- |
| K > 80      | 超買區 | 短期漲幅過大，需留意回檔風險 |
| 20 < K < 80 | 中性區 | 趨勢持續中，無極端信號       |
| K < 20      | 超賣區 | 短期跌幅過大，可能出現反彈   |

> ⚠️ **注意**：與 RSI 相同，在強勢趨勢中，KD 可能長期停留在超買或超賣區，稱為「鈍化」。此時不宜單純以超買超賣做反向操作。

### 金叉與死叉

- **金叉（Golden Cross）**：K 線由下往上穿越 D 線 → 看多信號
  - 在超賣區（20 以下）發生的金叉：強多信號
  - 在中性區發生的金叉：弱多信號
- **死叉（Death Cross）**：K 線由上往下穿越 D 線 → 看空信號
  - 在超買區（80 以上）發生的死叉：強空信號
  - 在中性區發生的死叉：弱空信號

### 背離（Divergence）

KD 背離是重要的反轉預警信號：

- **頂背離（Bearish Divergence）**：股價創新高，但 K 值未同步創新高 → 上漲動能減弱，可能即將反轉向下
- **底背離（Bullish Divergence）**：股價創新低，但 K 值未同步創新低 → 下跌動能減弱，可能即將反轉向上

> 📌 **實戰經驗**：連續出現兩次以上的背離信號比單次背離更為可靠。

### KD 鈍化處理

在強勢趨勢中 KD 會長時間停留在超買或超賣區，此時：

- **高檔鈍化**：K 值持續在 80 以上 → 強多頭趨勢，不宜過早放空
  - 應等待 K 值明確跌破 80 並出現死叉後再考慮減碼
- **低檔鈍化**：K 值持續在 20 以下 → 強空頭趨勢，不宜過早抄底
  - 應等待 K 值明確站上 20 並出現金叉後再考慮進場

## 進階應用

### KD 搭配均線

- KD 金叉 + 股價站上 20MA → 確認進場信號
- KD 死叉 + 股價跌破 20MA → 確認出場信號
- 均線方向為主（趨勢），KD 為輔（時機點）

### 多週期分析

- **週 KD**：判斷中期趨勢方向
- **日 KD**：尋找短線進出場點
- 策略：週 KD 方向為主，日 KD 配合進出場

### 與 RSI 的比較

| 比較項目 | KD               | RSI            |
| -------- | ---------------- | -------------- |
| 靈敏度   | 較高             | 較低           |
| 適用     | 短線波段         | 中線趨勢       |
| 鈍化     | 較容易鈍化       | 較不容易鈍化   |
| 主要信號 | 金叉/死叉        | 超買/超賣/背離 |
| 參考位置 | 收盤價在區間位置 | 漲跌幅比例     |

## 使用場景與限制

### 適用場景

- **盤整市場**：KD 指標在區間震盪時效果最好
- **短線交易**：KD 反應靈敏，適合短線波段操作
- **搭配其他指標**：金叉/死叉搭配均線或成交量確認

### 限制

- **鈍化問題**：強趨勢中容易產生假信號
- **震盪洗刷**：盤整時可能頻繁出現金叉/死叉，增加交易成本
- **落後性**：基於歷史價格計算，無法預測突發事件

## 實戰範例

假設台積電日 KD 走勢：

1. K 值從 15 向上金叉 D 值（超賣區金叉）→ 強多信號
2. 同時股價站上 20MA → 雙重確認 → 考慮進場
3. 持股過程中 K 值升至 85 → 進入超買區，提高警覺但不急著賣
4. K 向下死叉 D 值（超買區死叉）→ 考慮獲利了結
