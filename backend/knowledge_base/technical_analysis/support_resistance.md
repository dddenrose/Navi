# 支撐與壓力 (Support & Resistance)

## 定義

支撐與壓力是技術分析中最基本且最重要的概念之一，指的是股價在特定價位區域會出現買賣力量失衡，導致價格難以繼續往同方向移動。

- **支撐（Support）**：股價下跌到某個價位區域時，買方力量增強，使價格停止下跌或反彈的價位
- **壓力（Resistance）**：股價上漲到某個價位區域時，賣方力量增強，使價格停止上漲或回落的價位

> 📌 **核心觀念**：支撐與壓力不是一條精確的線，而是一個**價格區間**。實戰中應將其視為一個「帶」而非一個「點」。

## 支撐壓力的形成原因

### 一、心理因素

- **套牢者解套心理**：投資人在某價位買進後股價下跌，當股價回升到買入價附近時傾向賣出解套 → 形成壓力
- **獲利者回補心理**：投資人在某價位賣出後股價上漲，當股價回落到賣出價附近時傾向買回 → 形成支撐
- **整數關卡心理**：投資人對整數價位（如 100、500、1000 元）有天然的心理錨定

### 二、供需因素

- **支撐形成**：某價位有大量買盤等待（需求 > 供給）→ 價格難以跌破
- **壓力形成**：某價位有大量賣盤等待（供給 > 需求）→ 價格難以突破
- **籌碼密集區**：大量成交集中在某個價格帶，該區域自然形成支撐或壓力

### 三、技術因素

- **歷史價位記憶**：市場參與者記住過去的重要高低點，在接近時做出反應
- **指標匯聚**：多個技術指標（均線、趨勢線、費波那契）在同一價位附近重疊，加強該區域的支撐或壓力效力

## 找出支撐壓力的方法

### 一、前高前低法（最基本）

利用歷史上的明顯高點和低點作為參考：

- **前波低點**：股價過去在此位置止跌反彈 → 未來的支撐位
- **前波高點**：股價過去在此位置遇阻回落 → 未來的壓力位

#### 判斷強度

| 條件             | 效果                           |
| ---------------- | ------------------------------ |
| 被測試越多次未破 | 支撐/壓力越強                  |
| 停留時間越長     | 籌碼越密集，支撐/壓力越強      |
| 伴隨大成交量     | 代表多空交戰激烈，該位置越重要 |
| 距離現在越近     | 參考價值越高                   |

### 二、整數關卡與心理價位

- **整數價位**：50、100、200、500、1000 元等整數關卡
- **大盤整數**：萬點（10000）、兩萬點（20000）等
- **原因**：投資人掛單和心理停損/停利常設在整數位

> 整數關卡的支撐壓力效果在權值股和大盤指數上特別明顯。

### 三、均線支撐壓力（動態）

移動平均線是最常用的動態支撐壓力參考：

| 均線          | 支撐場景                     | 壓力場景               |
| ------------- | ---------------------------- | ---------------------- |
| 5MA（週線）   | 股價回落碰觸後反彈           | 股價反彈碰觸後再次下跌 |
| 20MA（月線）  | 多頭回檔的常見支撐           | 空頭反彈的常見壓力     |
| 60MA（季線）  | 中期趨勢的重要分水嶺         | 中期趨勢的重要分水嶺   |
| 240MA（年線） | 長期多空分界，又稱「牛熊線」 | 長期多空分界           |

#### 均線支撐壓力原則

- **多頭市場（股價在均線上方）**：均線是支撐
- **空頭市場（股價在均線下方）**：均線是壓力
- **多條均線匯聚在同一區域**：支撐或壓力大幅增強

### 四、趨勢線

連接兩個以上的重要高點或低點所畫出的直線：

#### 上升趨勢線（支撐線）

- **畫法**：連接兩個以上的**波段低點**，由左下往右上延伸
- **用途**：股價回落碰觸趨勢線時提供支撐 → 潛在買點
- **失效**：股價跌破上升趨勢線 → 趨勢可能反轉

#### 下降趨勢線（壓力線）

- **畫法**：連接兩個以上的**波段高點**，由左上往右下延伸
- **用途**：股價反彈碰觸趨勢線時遇到壓力 → 潛在賣點
- **失效**：股價突破下降趨勢線 → 趨勢可能反轉

#### 趨勢線有效性判斷

| 條件         | 說明                                  |
| ------------ | ------------------------------------- |
| 觸及次數 ≥ 3 | 至少被 3 次測試過的趨勢線才較可靠     |
| 持續時間長   | 持續數週/數月的趨勢線比數天的更有效   |
| 角度適中     | 太陡的趨勢線容易被跌破，45 度左右最佳 |
| 搭配量能確認 | 跌破/突破時放量確認有效性             |

### 五、費波那契回撤（Fibonacci Retracement）

利用費波那契數列中的比率預測價格回撤的支撐壓力位。

#### 關鍵回撤比率

| 比率  | 意義                                           |
| ----- | ---------------------------------------------- |
| 23.6% | 淺幅回撤，強勢趨勢的正常修正                   |
| 38.2% | 常見回撤位，多頭趨勢中的第一道重要支撐         |
| 50.0% | 半數回撤，心理上的重要分界                     |
| 61.8% | 黃金比率，最重要的回撤位，跌破此位趨勢可能反轉 |
| 78.6% | 深幅回撤，趨勢瀕臨反轉的最後防線               |

#### 使用方法

1. **上漲回撤**：找出一段上漲行情的最低點 A 和最高點 B
2. 從 B 開始往下計算各回撤比率對應的價位
3. 這些價位就是潛在的支撐位

$$
\text{回撤價位} = B - (B - A) \times \text{回撤比率}
$$

#### 實戰範例

假設某股票從 100 元漲到 200 元後開始回檔：

| 回撤比率 | 計算                    | 支撐價位 |
| -------- | ----------------------- | -------- |
| 23.6%    | 200 − (200−100) × 0.236 | 176.4 元 |
| 38.2%    | 200 − (200−100) × 0.382 | 161.8 元 |
| 50.0%    | 200 − (200−100) × 0.500 | 150.0 元 |
| 61.8%    | 200 − (200−100) × 0.618 | 138.2 元 |

### 六、跳空缺口

跳空缺口常形成天然的支撐或壓力區：

| 缺口類型   | 說明                       | 支撐/壓力效果        |
| ---------- | -------------------------- | -------------------- |
| 突破缺口   | 突破盤整區後的跳空         | 強支撐/強壓力        |
| 逃逸缺口   | 趨勢中段的加速跳空         | 中等支撐/壓力        |
| 竭盡缺口   | 趨勢末段的跳空             | 弱，通常會被回補     |
| 除權息缺口 | 台股除權息造成的價格不連續 | 非技術性缺口，需注意 |

> 📌 **缺口回補**：「缺口必補」是常見的說法，但實際上強勢的突破缺口和逃逸缺口不一定會被回補。

## 支撐壓力互換原則

這是支撐壓力分析中最重要的概念之一：

> **支撐一旦被跌破，就會轉變為壓力；壓力一旦被突破，就會轉變為支撐。**

### 原理

- 股價跌破原本的支撐位 → 在該位置買進的投資人變成套牢 → 當股價反彈接近該位置時，套牢者想解套賣出 → 原支撐變壓力
- 股價突破原本的壓力位 → 在該位置放空或賣出的投資人面對虧損 → 壓力消失並因追買而轉為支撐

### 判斷跌破/突破是否有效

| 條件            | 說明                                    |
| --------------- | --------------------------------------- |
| 收盤價確認      | 盤中短暫突破/跌破不算，需以收盤價確認   |
| 幅度 > 3%       | 超過支撐/壓力位 3% 以上較為可靠         |
| 連續 2-3 天維持 | 突破後在新位置穩定 2-3 天才算有效       |
| 成交量配合      | 放量突破/跌破較為可靠，縮量可能是假突破 |

## 實戰應用：用支撐壓力判斷進出場

### 進場策略

#### 策略一：支撐位買進

1. 找出下方最近的支撐位（前低、均線、費波那契）
2. 等待股價回落至支撐區附近
3. 觀察是否出現止跌信號（K 線反轉型態、KD 金叉、放量反彈）
4. 進場買入，停損設在支撐位下方 2-3%

#### 策略二：突破壓力位買進

1. 找出上方最近的壓力位（前高、下降趨勢線）
2. 等待股價帶量突破壓力位
3. 確認突破有效（收盤站穩、不跌回壓力位下方）
4. 進場買入，停損設在原壓力位（現轉為支撐位）下方

### 賣出策略

#### 策略一：壓力位獲利了結

1. 找出上方最近的壓力位
2. 股價接近壓力區時考慮分批獲利了結
3. 若股價在壓力位附近出現反轉信號（長上影線、KD 死叉）→ 減碼或出場

#### 策略二：跌破支撐位停損

1. 持有部位設定支撐位為停損參考
2. 股價跌破支撐位（收盤價確認）→ 執行停損
3. 不因「想等反彈」而忽略停損紀律

### 設定目標價

利用支撐壓力位設定合理的風險報酬比：

$$
\text{風險報酬比} = \frac{\text{進場價到目標壓力位的距離}}{\text{進場價到停損支撐位的距離}}
$$

- 風險報酬比 ≥ 2:1 才值得進場
- 例如：進場價 100 元，支撐停損 95 元（風險 5 元），壓力目標 115 元（報酬 15 元）→ 風報比 = 3:1 ✅

## 多重確認提高勝率

單一支撐壓力位的參考價值有限，當多個方法指向同一價格區間時，該區域的可靠度大幅提升：

### 支撐/壓力匯聚區（Confluence Zone）

當以下多種因素在同一價位附近重疊時，形成強力支撐或壓力：

- 前波低點/高點在 150 元附近
- 60 日均線目前在 148 元
- 費波那契 38.2% 回撤位在 151 元
- 上升趨勢線通過 149 元

→ 148-151 元區間形成**強力支撐匯聚區**，在此進場的勝率較高

> 📌 **經驗法則**：至少有 2-3 個不同方法的支撐/壓力重疊才算有效的匯聚區。

## 使用場景與限制

### 適用場景

- **判斷進場時機**：在支撐區附近尋找買點
- **設定停損停利**：用壓力位設目標價，用支撐位設停損
- **評估風險報酬比**：計算進場的合理性
- **搭配其他指標**：先用支撐壓力找價位，再用 KD/RSI/量能確認信號

### 限制

- **非精確價位**：支撐壓力是區間概念，不要期望精確到某個價格
- **會被突破**：所有支撐壓力都有被突破的可能，必須設定停損
- **主觀性**：不同人畫出的趨勢線可能不同
- **失效場景**：重大消息（財報爆雷、政策變動）可能直接穿越所有支撐壓力
- **不適用無量股**：成交量過低的股票，支撐壓力的參考性降低
