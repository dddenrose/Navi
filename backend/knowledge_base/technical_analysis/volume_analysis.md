# 成交量分析 (Volume Analysis)

## 定義

成交量是指在特定時間內股票被買賣的總數量（股數或張數），是技術分析中唯一獨立於價格之外的市場數據。成交量反映市場的參與熱度與資金流動情況，被稱為「股市的燃料」。

> 📌 **經典格言**：「量是價的先行指標」——成交量往往領先價格變化。

## 基本概念

### 量能指標

| 指標   | 說明                                       |
| ------ | ------------------------------------------ |
| 成交量 | 單日（或單期）買賣股數                     |
| 成交額 | 成交量 × 成交價，反映真實資金流入量        |
| 均量線 | N 日成交量的移動平均，判斷量能趨勢         |
| 量比   | 當日成交量 ÷ 過去 5 日平均量，判斷量能變化 |
| 換手率 | 成交量 ÷ 流通股數 × 100%                   |

### 量能水準判斷

| 換手率   | 意義                                 |
| -------- | ------------------------------------ |
| < 1%     | 極度冷清，無人關注                   |
| 1% ~ 3%  | 正常交投                             |
| 3% ~ 7%  | 交投活絡，受市場關注                 |
| 7% ~ 15% | 高度活躍，可能有主力資金進出         |
| > 15%    | 異常活躍，需特別留意（可能為消息面） |

## 量價關係判讀

### 核心原則

量價關係是技術分析中最基礎且重要的概念。透過觀察「量」與「價」的配合程度，判斷趨勢的真實性與持續性。

### 八種量價關係

#### 健康型態（趨勢延續信號）

| 型態     | 量     | 價     | 意義                             |
| -------- | ------ | ------ | -------------------------------- |
| 量增價漲 | 放大 ↑ | 上漲 ↑ | 多頭強勢，買氣旺盛，趨勢健康     |
| 量縮價跌 | 萎縮 ↓ | 下跌 ↓ | 賣壓減輕，空頭動能不足，可能止跌 |
| 量縮價平 | 萎縮 ↓ | 持平 → | 籌碼沉澱，盤整蓄力               |

#### 警示型態（趨勢可能反轉）

| 型態     | 量     | 價     | 意義                         |
| -------- | ------ | ------ | ---------------------------- |
| 量增價跌 | 放大 ↑ | 下跌 ↓ | 賣壓沉重，多方無力承接，偏空 |
| 量縮價漲 | 萎縮 ↓ | 上漲 ↑ | 追價意願低，上漲可能難以持續 |
| 量增價平 | 放大 ↑ | 持平 → | 多空分歧大，注意方向選擇     |

#### 極端型態

| 型態     | 量       | 價   | 意義                           |
| -------- | -------- | ---- | ------------------------------ |
| 爆量長紅 | 急劇放大 | 大漲 | 可能為突破信號，也可能是末升段 |
| 爆量長黑 | 急劇放大 | 大跌 | 恐慌性賣壓，可能為最後一跌     |

## 成交量型態分析

### 一、底部放量

- **特徵**：股價在低檔盤整多時，突然出現明顯放量
- **意義**：可能有大資金開始佈局，底部可能成形
- **確認**：後續需持續出現量增價漲才能確認

### 二、頂部放量

- **特徵**：股價在高檔區出現創新高的成交量，但股價漲幅有限或收長上影線
- **意義**：多方力竭，主力可能在高檔出貨
- **確認**：若隨後出現量縮無法再創新高 → 頭部確認

### 三、突破放量

- **特徵**：股價突破重要壓力位（均線、前高、型態頸線）時伴隨放量
- **意義**：突破有效性高，趨勢可能延續
- **警告**：突破時若量能不足（縮量突破），容易變成假突破

### 四、地量

- **特徵**：成交量萎縮至近期（3-6 個月）最低水準
- **意義**：「地量見地價」——市場極度冷清，往往是底部即將出現的徵兆
- **注意**：地量不一定馬上反轉，可能持續一段時間的低量盤整

### 五、天量

- **特徵**：成交量創近期（3-6 個月）最高水準
- **意義**：「天量見天價」——多空劇烈換手，可能是趨勢轉折點
- **區分**：需配合價格判斷是突破天量（正面）還是出貨天量（負面）

## 均量線應用

均量線與均價線（MA）類似，用於判斷量能的趨勢方向。

### 常用週期

| 均量線 | 用途         |
| ------ | ------------ |
| 5 日   | 短線量能變化 |
| 20 日  | 中期量能趨勢 |
| 60 日  | 長期量能基準 |

### 判讀方式

- **5 日均量 > 20 日均量**：短期量能增溫，交投轉趨活絡
- **5 日均量 < 20 日均量**：短期量能降溫，交投轉趨清淡
- **均量排列向上**：資金持續流入，有利多方
- **均量排列向下**：資金持續流出，有利空方

## 搭配其他指標

### 量 + 均線

- 股價站上均線 + 放量 → 突破有效，偏多
- 股價跌破均線 + 放量 → 跌破有效，偏空
- 股價站上均線 + 縮量 → 假突破風險高

### 量 + KD / RSI

- KD 金叉 + 放量 → 做多信號增強
- RSI 超賣 + 放量反轉 → 底部反彈機率高
- 指標翻多但量能不配合 → 信號可靠度降低

### 量 + 布林通道

- 帶寬收縮後放量突破上軌 → 強烈做多信號
- 縮量跌破下軌 → 可能是恐慌性拋售尾聲

## 使用場景與限制

### 適用場景

- **確認趨勢真偽**：量價配合是判斷趨勢是否真實的核心方法
- **辨識轉折點**：異常量能常出現在行情轉折之前
- **搭配其他指標**：為價格指標的信號提供確認依據

### 限制

- **無方向性**：成交量本身不區分買賣方向
- **受消息面影響**：重大新聞、除權息日等會造成非趨勢性的量能變化
- **個股差異大**：權值股與小型股的量能規模不同，需以換手率或量比做標準化比較
- **高頻交易影響**：程式交易可能產生非自然的成交量波動
