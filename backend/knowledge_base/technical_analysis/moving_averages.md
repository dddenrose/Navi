# 均線理論 — 移動平均線 (Moving Averages)

## 定義

移動平均線（Moving Average, MA）是將一段時間內的收盤價加以平均後連線而成的技術指標，是最基本也最重要的趨勢追蹤工具。它能平滑價格波動，幫助投資人辨識趨勢方向。

## 均線類型

### 簡單移動平均線 SMA (Simple Moving Average)

$$
SMA_N = \frac{P_1 + P_2 + \cdots + P_N}{N}
$$

- 每一天的價格權重相同
- 計算簡單直觀
- 缺點：對近期價格變動反應較慢

### 指數移動平均線 EMA (Exponential Moving Average)

$$
EMA_t = P_t \times k + EMA_{t-1} \times (1 - k), \quad k = \frac{2}{N + 1}
$$

- 近期價格權重較高，對新資訊反應更快
- 比 SMA 更能即時反映趨勢變化
- 適合較短週期的交易策略

### SMA vs EMA 比較

| 特性     | SMA        | EMA      |
| -------- | ---------- | -------- |
| 計算方式 | 等權重平均 | 指數加權 |
| 靈敏度   | 較低       | 較高     |
| 假信號   | 較少       | 較多     |
| 適用     | 中長線趨勢 | 短線交易 |

## 常用均線週期

| 均線  | 週期   | 代表意義                   |
| ----- | ------ | -------------------------- |
| 5MA   | 5 日   | 一週線，極短線趨勢         |
| 10MA  | 10 日  | 兩週線，短線趨勢           |
| 20MA  | 20 日  | 月線，短中線趨勢（最常用） |
| 60MA  | 60 日  | 季線，中線趨勢             |
| 120MA | 120 日 | 半年線，中長線趨勢         |
| 240MA | 240 日 | 年線，長線趨勢（牛熊分界） |

## 核心判讀方法

### 1. 均線方向

- **均線向上**：趨勢看多
- **均線走平**：趨勢盤整
- **均線向下**：趨勢看空

### 2. 價格與均線的關係

- **股價在均線之上**：多頭格局
- **股價在均線之下**：空頭格局
- **股價接近均線**：可能出現支撐或壓力

### 3. 黃金交叉 (Golden Cross)

短期均線**由下往上穿越**長期均線，代表短期趨勢轉強。

- **經典組合**：5MA 上穿 20MA、20MA 上穿 60MA
- **強勢信號**：交叉後兩條均線都向上
- **弱勢信號**：交叉後長期均線仍走平或向下（可能只是反彈）

### 4. 死亡交叉 (Death Cross)

短期均線**由上往下穿越**長期均線，代表短期趨勢轉弱。

- **經典組合**：5MA 下穿 20MA、20MA 下穿 60MA
- **確認方式**：交叉後量能放大，信號更可靠

### 5. 多頭排列與空頭排列

- **多頭排列**：5MA > 20MA > 60MA > 120MA，所有均線向上 → 強勢上漲趨勢
- **空頭排列**：5MA < 20MA < 60MA < 120MA，所有均線向下 → 強勢下跌趨勢
- **糾結（纏繞）**：多條均線交纏在一起 → 盤整，即將選擇方向

### 6. 葛蘭碧八大法則 (Granville's Rules)

Joseph Granville 提出的均線操作八大法則：

**買入信號（4 個）：**

1. 均線從下降轉為走平或上升，股價從下方突破均線 → 買入
2. 股價跌破均線後隨即回到均線之上，且均線持續上升 → 買入
3. 股價在均線之上回落但未跌破均線即反轉上升 → 買入（均線支撐）
4. 股價跌至均線下方且遠離均線，出現急跌 → 短線搶反彈

**賣出信號（4 個）：** 5. 均線從上升轉為走平或下降，股價從上方跌破均線 → 賣出 6. 股價突破均線後隨即回落到均線之下，且均線持續下降 → 賣出 7. 股價在均線之下反彈但未突破均線即反轉下跌 → 賣出（均線壓力）8. 股價漲至均線上方且遠離均線，出現急漲 → 短線獲利了結

## 實戰技巧

### 均線乖離率

當股價偏離均線太遠時，有回歸均線的傾向：

$$
乖離率 = \frac{收盤價 - MA}{MA} \times 100\%
$$

- 乖離率過高（正值）：股價超漲，有回檔風險
- 乖離率過低（負值）：股價超跌，有反彈機會

### 均線扣抵

均線的方向受到「即將被扣除的舊數據」影響。如果即將扣除的是一個低價，均線可能繼續上升；如果扣除的是高價，均線可能轉向下降。這稱為「扣抵」。

## 限制

- **滯後指標**：均線本質上是滯後的，只能確認趨勢而非預測趨勢
- **盤整行情**：在無趨勢的盤整行情中，均線交叉頻繁出現假信號
- **參數選擇**：不同商品、不同市場可能需要不同的均線參數
