# 布林通道 (Bollinger Bands)

## 定義

布林通道由 John Bollinger 於 1980 年代發明，是一種利用統計學標準差概念建構的價格通道指標。由三條線組成，能夠動態反映價格的波動程度，是判斷股價相對高低位和波動率變化的重要工具。

## 組成元素

| 元素   | 計算方式                   | 意義             |
| ------ | -------------------------- | ---------------- |
| 中軌線 | N 日簡單移動平均線（SMA）  | 趨勢方向的基準線 |
| 上軌線 | 中軌 + K × 標準差          | 價格的上方壓力帶 |
| 下軌線 | 中軌 − K × 標準差          | 價格的下方支撐帶 |
| 帶寬   | (上軌 − 下軌) ÷ 中軌 × 100 | 波動率大小       |

## 計算公式

### 步驟一：計算中軌線（SMA）

$$
\text{中軌} = SMA(N) = \frac{1}{N} \sum_{i=1}^{N} C_i
$$

### 步驟二：計算標準差

$$
\sigma = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (C_i - SMA)^2}
$$

### 步驟三：計算上下軌

$$
\text{上軌} = SMA(N) + K \times \sigma
$$

$$
\text{下軌} = SMA(N) - K \times \sigma
$$

### 步驟四：計算帶寬（Bandwidth）

$$
BW = \frac{\text{上軌} - \text{下軌}}{\text{中軌}} \times 100
$$

### 步驟五：計算 %B 指標

$$
\%B = \frac{C - \text{下軌}}{\text{上軌} - \text{下軌}}
$$

> %B 表示收盤價在通道中的相對位置：1 = 上軌，0 = 下軌，0.5 = 中軌。

### 標準參數

| 參數     | 常用值 | 說明                                      |
| -------- | ------ | ----------------------------------------- |
| 週期 (N) | 20     | Bollinger 建議值，約一個月的交易日        |
| 倍數 (K) | 2      | 涵蓋約 95.4% 的價格分佈（常態分佈假設下） |

> 📌 保守型投資人可使用 K=2.5（涵蓋更多價格），積極型可用 K=1.5（更窄通道）。

## 判讀方法

### 一、通道位置判斷

| 價格位置         | 意義                               |
| ---------------- | ---------------------------------- |
| 觸及或突破上軌   | 短期偏強，可能超漲（但不一定反轉） |
| 在中軌與上軌之間 | 偏多走勢                           |
| 在中軌附近       | 中性，觀望                         |
| 在中軌與下軌之間 | 偏空走勢                           |
| 觸及或跌破下軌   | 短期偏弱，可能超跌（但不一定反彈） |

> ⚠️ **重要觀念**：觸及上軌不代表一定要賣，觸及下軌也不代表一定要買。在強趨勢中，價格會沿著上軌（或下軌）持續行進，稱為「走通道」。

### 二、通道寬度變化（波動率分析）

布林通道最核心的價值在於**通道寬度的變化**：

#### 通道收縮（Squeeze）

- 帶寬縮窄到歷史低點 → **盤整蓄力階段**
- 代表市場處於低波動狀態，大行情即將來臨
- 是布林通道最重要的信號之一：「寧靜之後必有風暴」

#### 通道擴張（Expansion）

- 帶寬急速放大 → **趨勢行情啟動**
- 價格突破上軌 + 通道擴張 = 向上趨勢確認
- 價格跌破下軌 + 通道擴張 = 向下趨勢確認

#### 通道收斂（Contraction）

- 帶寬從高點開始縮小 → **趨勢可能結束**
- 通常出現在一波行情走完之後

### 三、經典交易策略

#### 策略一：通道突破策略

適用於盤整後的突破行情：

1. 等待帶寬收縮至近 6 個月最窄
2. 價格放量突破上軌 → 做多
3. 價格放量跌破下軌 → 做空
4. 停損設在中軌

#### 策略二：均值回歸策略

適用於區間震盪市場：

1. 價格觸及上軌且 RSI > 70 → 考慮減碼
2. 價格觸及下軌且 RSI < 30 → 考慮加碼
3. 目標價為中軌線
4. 停損設在上/下軌外一定比例

#### 策略三：走通道策略（Riding the Band）

適用於強趨勢行情：

1. 價格持續沿上軌行進 → 持有多單
2. 跌破中軌 → 考慮出場
3. 反之亦然（空頭走下軌）

### 四、搭配其他指標

- **布林 + RSI**：觸及上軌 + RSI 超買 = 雙重警示；觸及下軌 + RSI 超賣 = 強力支撐信號
- **布林 + 成交量**：突破上/下軌時需搭配放量確認，縮量突破易為假突破
- **布林 + MACD**：帶寬收縮 + MACD 金叉 = 向上突破機率較高

## 使用場景與限制

### 適用場景

- **波動率分析**：判斷市場從盤整轉為趨勢的轉折點
- **支撐壓力參考**：動態的支撐與壓力帶
- **風控輔助**：利用通道寬度設定停損/停利位

### 限制

- **常態分佈假設**：真實股價分佈有厚尾（fat tails），極端行情下通道可能失效
- **落後指標**：以歷史數據計算，無法預測突發事件
- **無法獨立使用**：需搭配成交量或其他指標確認信號
- **參數敏感**：不同週期和倍數會產生截然不同的通道

## 實戰範例

假設某股票走勢：

1. 連續兩個月帶寬持續收縮 → 盤整蓄力，密切關注
2. 某日放量突破上軌，帶寬急速擴張 → 向上突破確認
3. 後續股價沿上軌持續上行 → 持有，以中軌為保護性停損
4. 股價回落碰觸中軌後反彈 → 中軌支撐有效，可加碼
5. 股價跌破中軌 + 帶寬開始收縮 → 趨勢結束，考慮出場
